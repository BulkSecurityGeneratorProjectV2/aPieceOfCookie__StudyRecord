红黑树的特点：

1. 根节点必须是黑色

2. 每个叶子节点（NIL）都是黑色的空节点

3. 从根节点到叶子节点，不能出现两个连续的红色节点

4. 从任一节点出发，到它下边的子节点的路径包含的黑色节点数目都相同
